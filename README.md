# 📧 Automated Email Reporting System

## 🔍 Overview

This project automates daily performance reporting by sending emails to State Heads and Branch Heads. It eliminates manual reporting and ensures timely delivery of insights.

---

## 🛠 Tools Used

* Python
* Pandas
* SMTP / Email Automation
* Excel / Data Source

---

## 📂 Project Files

* `state_head_mail.py` → Sends reports to State Heads
* `branch_head_mail.py` → Sends reports to Branch Heads

---

## ⚙️ Features

* Automated email sending
* Dynamic recipient mapping
* Data-driven reporting using DataFrame
* Daily performance summary

---

## 📊 Business Impact

* Reduced manual reporting effort
* Faster decision-making
* Improved reporting accuracy
